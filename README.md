# Touch Bar if err

easy and fast insert 
```
if err != nil {
  return
}
```

<img src="https://raw.githubusercontent.com/dongri/touch-bar-if-err/master/media/touch-bar.png" alt="Touch Bar" title="Touch Bar">

## Installation

```
$ code --install-extension D.touch-bar-if-err
```
